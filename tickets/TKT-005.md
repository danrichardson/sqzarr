## Description
Adding multiple directories with the same settings (min age, max bitrate, min size) is tedious — each one must be configured individually. The user wants a way to either:

1. **Copy/preset**: Copy settings from an existing directory config and apply them to multiple new directories at once (like a preset/template).
2. **Subdirectory support**: Add multiple subdirectories within a parent that inherit the parent's settings.

For example, `/media/Videos/Backlog` has settings (min age: 7, max bitrate: 2222000, min size: 500MB). The user wants to quickly apply those same settings to many other directories without re-entering each field every time.

This has been requested before and not implemented.
