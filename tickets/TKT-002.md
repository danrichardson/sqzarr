## Description
Several stat blocks on the main dashboard are currently static and should be interactive:

1. **Jobs Done block** — should be clickable and show a breakdown of how that number was calculated (list of completed jobs with details).
2. **Failed block** — should be clickable and navigate to the failed items so the user can see what failed and why.
3. **Status block** — currently shows a start/stop control, but there's also a separate pause button. The purpose and distinction between these is unclear. Investigate whether both are needed or if one should be removed/consolidated.
